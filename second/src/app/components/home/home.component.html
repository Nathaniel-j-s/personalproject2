MEAN Stack

Because clearly, I'm not the right kind of person for a SANE stack. I'm using MongoDB, Express, Angular 2 (look upon what you have wrought, Jeremy!), and Node.js. It may be complicated, but it's super awesome and I'm proud of it.

Angular CLI

The Angular CLI has been used to generate my components, services, etc. This makes for easy compilation (with Webpack (20 points, oh yeah (up to 60, if you're keeping track))).

JWT Tokens

Full featured authentication using JSON web tokens. Login and store user data. I did not do this part myself. I followed Brett's advice and let someone else do the authentication stuff. I just copied it.
